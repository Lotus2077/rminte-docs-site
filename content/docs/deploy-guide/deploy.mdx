## Service Provider Deployment Guide

As a service provider, you are the bridge connecting RM-01 technology with the actual needs of enterprises. This section will guide you on how to efficiently deploy RM-01 devices for enterprise clients and provide related technical support.

### 1. Preparation Work

<Steps>
  <Step>
    #### Device and Accessories Verification
    
    Before delivery to the enterprise, please ensure the following items are complete:
    
    <div className="p-4 border rounded-lg">
      <strong>RM-01 Accessory Package</strong>
      <ul className="list-disc pl-6 mt-2">
        <li>RM-01 Main Device</li>
        <li>CFexpress Type B Storage Card (with pre-installed models & applications)</li>
        <li>MicroSD Card (optional, for expanded storage)</li>
        <li>USB-C Power Adapter (PD3.1, up to 140W)</li>
        <li>USB-C to Ethernet Adapter</li>
        <li>Quick Start Guide</li>
        <li>Warranty Card</li>
      </ul>
    </div>
  </Step>
  
  <Step>
    #### Model and Application Preparation
    
    Prepare suitable models and applications according to enterprise needs:
    
    - **Standard Model Package**: Contains commonly used open-source models such as DeepSeek, Qwen, Llama etc.
    - **Industry-Specific Model Package**: Select appropriate vertical domain models based on the enterprise's industry
    - **Enterprise Custom Models**: If the enterprise has their own trained models, ensure they are converted to formats suitable for running on RM-01
  </Step>
  
  <Step>
    #### Enterprise Environment Assessment
    
    Conduct an environment assessment before deployment:
    
    - **Network Environment**: Although RM-01 supports offline operation, initial setup and updates may require network connectivity
    - **Power Supply**: Ensure stable power supply at the deployment location
    - **Physical Security**: Assess the physical security measures of the device placement location
    - **IT Infrastructure**: Understand the enterprise's existing IT infrastructure to ensure smooth integration of RM-01
  </Step>
</Steps>

### 2. Hardware Deployment Process

<Steps>
  <Step>
    #### Device Installation
    
    1. Choose an appropriate location for the RM-01 (desktop or dedicated cabinet)
    2. Connect the power adapter
    3. If wired network is needed, connect the USB-C to Ethernet adapter
    4. Insert the CFexpress storage card pre-loaded with required models
    5. Insert the MicroSD card pre-loaded with applications (if applicable)
    
    <Callout type="warning">
      Ensure the device is placed in a well-ventilated environment and avoid blocking the cooling vents.
    </Callout>
  </Step>
  
  <Step>
    #### Startup and Initialization
    
    1. Press the power button to turn on the device
    2. The device will automatically initialize and load models and applications from the storage card (takes about 5 minutes)
    3. During initialization, you can connect a laptop via USB-C port to monitor the device status
  </Step>
  
  <Step>
    #### Network Configuration
    
    1. Connect to the enterprise LAN through the USB-C network interface
    2. Configure network parameters:
       - IP address (static/DHCP)
       - Subnet mask
       - Gateway
       - DNS servers
    3. Verify network connectivity
  </Step>
</Steps>
