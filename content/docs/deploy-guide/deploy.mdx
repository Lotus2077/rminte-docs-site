## Service Provider Deployment Process

### 1. Preparation Work

<Steps>
  <Step>
    #### Model and Application Preparation
    
    Prepare suitable models and applications according to enterprise needs:
    
    - **Standard Model Package**: Contains commonly used open-source models such as DeepSeek, Zhipu, Qwen, etc.
    - **Industry-Specific Model Package**: Select appropriate vertical domain models based on the enterprise's industry
    - **Enterprise Custom Models**: If the enterprise has their own trained models, ensure they are converted to formats suitable for running on RM-01 and stored in the specified directory on the CFexpress card
    - **Enterprise Custom Applications**: AI applications developed using Dify or other development tools based on the RMinte platform, stored as encrypted Docker Images in the specified directory on the CFexpress card
  </Step>

  <Step>
    #### Enterprise Environment Assessment
    
    Conduct an environment assessment before deployment:
    
    - **Network Environment**: Evaluate the feasibility of connecting RM-01 to the enterprise's internal network according to client requirements
    - **Power Supply**: Ensure stable power supply at the deployment location
    - **Physical Security**: Assess the physical security of the device placement location and develop corresponding security measures
    - **IT Infrastructure**: Understand the enterprise's existing IT infrastructure to ensure smooth integration of RM-01
  </Step>
  
  <Step>
    #### Device and Accessories Verification
    
    Before delivery to the enterprise, please ensure the following items are complete:
    
    <div className="p-4 border rounded-lg">
      <strong>RM-01 Device Package</strong>
      <ul className="list-disc pl-6 mt-2">
        <li>RM-01 Main Device</li>
        <li>CFexpress Type B Storage Card (pre-installed with models and applications)</li>
        <li>Quick Start Guide</li>
        <li>Warranty Card</li>
      </ul>
      <strong>RM-01 Accessory Package</strong>
      <ul className="list-disc pl-6 mt-2">
        <li>MicroSD Card (optional, for encryption)</li>
        <li>USB-C Power Adapter (PD3.1, up to 140W)</li>
        <li>USB-C to Ethernet Adapter</li>
      </ul>
    </div>
  </Step>
</Steps>

### 2. Hardware Deployment Process

<Steps>
  <Step>
    #### Device Installation
    
    1. Choose an appropriate location for the RM-01 (desktop or dedicated cabinet)
    2. Insert the CFexpress storage card pre-loaded with required models and applications
    3. Insert the MicroSD card for encryption (if applicable)
    4. Connect the power adapter
    5. If wired network is needed, connect the USB-C to Ethernet adapter
    
    <Callout type="warning">
      Ensure the device is placed in a well-ventilated environment and avoid blocking the cooling vents.
    </Callout>
  </Step>
  
  <Step>
    #### Startup and Initialization
    
    1. Press the power button to turn on the device
    2. The device will automatically initialize and load models and applications from the storage card (takes about 5 minutes)
    3. After 5 minutes, RM-01 will complete loading models and applications and enter the ready state

    <Callout type="info">
      During initialization, you can connect a laptop via USB-C port to monitor the device status
    </Callout>
  </Step>
  
  <Step>
    #### Network Configuration
    
    1. Connect to the enterprise LAN through the USB-C port (with adapter if needed)
    2. Configure network parameters:
       - IP address (static/DHCP)
       - Subnet mask
       - Gateway
       - DNS servers
    3. Verify network connectivity
  </Step>
</Steps>
