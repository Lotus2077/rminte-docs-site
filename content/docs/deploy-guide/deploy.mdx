## Service Provider Deployment Process

### 1. Pre-deployment Preparation

<Steps>
  <Step>
    #### Model and Application Preparation
    
    Prepare suitable models and applications based on enterprise requirements:
    
    - **Standard Model Package**: Includes common open-source models such as DeepSeek, Zhipu, Qwen, etc.
    - **Industry-specific Model Package**: Select vertical domain models suitable for the enterprise's industry
    - **Enterprise Custom Models**: If the enterprise has its own trained models, ensure they are converted to formats suitable for RM-01 and stored in the specified directory on the CFexpress card
    - **Enterprise Custom Applications**: AI applications developed using Dify or other development tools based on the RMinte platform, stored as encrypted Docker Images in the specified directory on the CFexpress card
  </Step>

  <Step>
    #### Enterprise Environment Assessment
    
    Conduct environment assessment before deployment:
    
    - **Network Environment**: Evaluate the feasibility of connecting RM-01 to the enterprise internal network according to customer requirements
    - **Power Supply**: Ensure stable power supply at the deployment location
    - **Physical Security**: Assess the physical security of the device placement location and develop corresponding security measures
    - **IT Infrastructure**: Understand the enterprise's existing IT infrastructure to ensure smooth integration of RM-01
  </Step>
  
  <Step>
    #### Device and Accessories Verification
    
    Before delivery to the enterprise, ensure the following items are complete:
    
    <div className="p-4 border rounded-lg">
      <strong>RM-01 Device Package</strong>
      <ul className="list-disc pl-6 mt-2">
        <li>RM-01 Host</li>
        <li>CFexpress Type B Storage Card (pre-installed with models and applications)</li>
        <li>MicroSD Card</li>
        <li>Quick Start Guide</li>
        <li>Warranty Card</li>
      </ul>
      <strong>RM-01 Accessories Package</strong>
      <ul className="list-disc pl-6 mt-2">
        <li>USB-C Power Adapter (PD3.1, up to 140W)</li>
        <li>USB-C to Ethernet Adapter</li>
      </ul>
    </div>
  </Step>
</Steps>

### 2. Hardware Deployment Process

<Steps>
  <Step>
    #### Device Installation
    
    1. Choose a suitable location for RM-01 (desktop or dedicated cabinet)
    2. Insert the CFexpress storage card pre-installed with required models and applications
    3. Insert the MicroSD card
    4. Connect the power adapter
    5. If wired network is needed, connect the USB-C to Ethernet adapter
    
    <Callout type="warning">
      Ensure the device is placed in a well-ventilated environment, avoiding obstruction of heat dissipation vents.
    </Callout>
  </Step>
  
  <Step>
    #### Startup and Initialization
    
    1. Press the power button to start
    2. The device will automatically initialize and load models and applications from the storage card (approximately 5 minutes)
    3. After 5 minutes, RM-01 will complete loading models and applications and enter a ready-to-use state

    <Callout type="info">
      During initialization, you can connect a laptop via USB-C port to monitor device status
    </Callout>
  </Step>
  
  <Step>
    #### Network Configuration
    
    1. Connect to the enterprise LAN through the USB-C port (with adapter if needed)
    2. Configure network parameters:
       - IP Address (Static/DHCP)
       - Subnet Mask
       - Gateway
       - DNS Servers
    3. Verify network connection
  </Step>
</Steps>
