## Service Provider Deployment and Delivery Process

The deployment process consists of 5 main phases designed to ensure smooth implementation and successful delivery to enterprise customers.

<Steps>
  <Step>
    ### Pre-deployment Preparation

    Prepare models, assess enterprise environment, and verify device package contents.

    **Model and Application Preparation**
    - **Standard Model Package**: Common open-source models (DeepSeek, Zhipu, Qwen)
    - **Industry-specific Model Package**: Vertical domain models for the enterprise's industry
    - **Enterprise Custom Models**: Customer-trained models converted for RM-01
    - **Enterprise Custom Applications**: AI applications developed using Dify or other tools

    **Enterprise Environment Assessment**
    - **Network Environment**: Evaluate RM-01 connectivity to internal network
    - **Power Supply**: Ensure stable power at deployment location
    - **Physical Security**: Assess device placement security
    - **IT Infrastructure**: Understand existing systems for smooth integration

    **Device and Accessories Verification**
    
    Before delivery to the enterprise, ensure the following items are complete:

    **RM-01 Device Package**
    - RM-01 Host
    - CFexpress Type B Storage Card (pre-installed with models and applications)
    - MicroSD Card
    - Quick Start Guide
    - Warranty Card

    **RM-01 Accessories Package**
    - USB-C Power Adapter (PD3.1, up to 140W)
    - USB-C to Ethernet Adapter
  </Step>
  
  <Step>
    ### Hardware Deployment Process

    Install, initialize, and configure the RM-01 device.

    **Device Installation**
    1. Choose suitable location (desktop or dedicated cabinet)
    2. Insert CFexpress storage card with models and applications
    3. Insert MicroSD card
    4. Connect power adapter
    5. Connect USB-C to Ethernet adapter if needed

    <Callout title="Installation Safety" type="warn">
      Ensure the device is placed in a well-ventilated environment, avoiding obstruction of heat dissipation vents.
    </Callout>

    **Startup and Initialization**
    1. Press power button to start
    2. Device automatically initializes (approximately 5 minutes)
    3. RM-01 enters ready-to-use state

    <Callout title="Monitoring Initialization">
      During initialization, you can connect a laptop via USB-C port to monitor device status and track the loading progress.
    </Callout>

    **Network Configuration**
    1. Connect to the enterprise LAN through the USB-C port (with adapter if needed)
    2. Configure network parameters:
       - IP Address (Static/DHCP)
       - Subnet Mask
       - Gateway
       - DNS Servers
    3. Verify network connection
  </Step>
  
  <Step>
    ### Verification and Testing

    Conduct comprehensive testing to ensure optimal performance.

    **Testing Areas**
    - **Functional Testing**: Model loading, application functionality, user authorization
    - **Performance Testing**: Inference speed, concurrent processing, stability
    - **Connectivity Testing**: System integration, API connections, external calls

    <Callout title="Testing Recommendation" type="success">
      It is recommended to conduct comprehensive functional benchmark testing based on the RM-01 testing checklist to ensure optimal device performance in the customer environment. Generate a test report upon completion as part of delivery documentation.
    </Callout>
  </Step>
  
  <Step>
    ### Training and Delivery

    Provide comprehensive training and deliver complete documentation.

    **Enterprise Administrator Training**
    1. **Device Operation and Maintenance** - Basic startup/shutdown and hardware maintenance
    2. **Management Interface Usage** - System settings and monitoring
    3. **Model and Application Management** - Updates and configuration
    4. **Troubleshooting** - Common issues diagnosis and resolution

    **End User Training**
    - **Application Access Methods**: How to log in and find needed applications
    - **Basic Operation Guide**: Common functions and operation flow demonstration
    - **Usage Tips and Best Practices**: Professional suggestions for improving efficiency
    - **Common Issues Resolution**: Handling operational errors and exceptions

    **Documentation Delivery**
    - **Device User Manual**: Hardware specifications and basic operation guide
    - **Administrator Guide**: System management and troubleshooting
    - **User Operation Guide**: Application usage steps and instructions
    - **Troubleshooting Guide**: Common issues and solutions

    <Callout title="Training Best Practice">
      It is recommended to record training videos for administrator reference and customize training content for different department users to improve training effectiveness.
    </Callout>
  </Step>
  
  <Step>
    ### After-sales Support

    Provide ongoing support and maintenance services.

    **Support Services**
    - **Remote Support**: Health checks, troubleshooting, 24/7 emergency support
    - **Updates & Maintenance**: Model updates, firmware assistance, optimization
    - **Extended Services**: Custom development, data integration, on-site support

    **Service Level Agreements**
    - **Standard Response**: 4-hour response during business hours
    - **Premium Response**: 2-hour response during business hours
    - **Emergency Response**: 30-minute response 24/7

    <Callout title="SLA Guarantee">
      All SLA commitments are backed by service quality guarantees with compensation for unmet standards. Flexible SLA options available, customizable based on enterprise requirements.
    </Callout>
  </Step>
</Steps>

## Frequently Asked Questions

<Accordions type="multiple">
  <Accordion title="Does RM-01 require continuous internet connection?">
    No. RM-01 is designed to operate completely offline, ensuring enterprise data never leaves the local environment.
  </Accordion>
  <Accordion title="What is RM-01's power consumption?">
    RM-01 has a maximum power consumption of 100W, with a TDP of 60W, saving 98.6% energy compared to traditional AI (single A100) servers. Annual electricity cost for 24/7 operation is approximately $480.
  </Accordion>
  <Accordion title="What model sizes does RM-01 support?">
    RM-01 supports models ranging from 0.5B to 235B (GPTQ Int4) parameters, covering mainstream open-source models including DeepSeek, Zhipu, Qwen, and many others.
  </Accordion>
  <Accordion title="Does the device require professional IT staff for maintenance?">
    No. RM-01 features zero-maintenance design, allowing regular users to easily use and manage it after basic training provided during deployment.
  </Accordion>
  <Accordion title="How to add new models to RM-01?">
    Store model files on the CFexpress storage card at the specified directory, insert it into the device, and the system will automatically recognize and load the models.
  </Accordion>
  <Accordion title="How is enterprise data kept confidential?">
    RM-01 uses hardware-level asymmetric encryption, with all data processed locally and never uploaded to the cloud, ensuring absolute enterprise data security.
  </Accordion>
  <Accordion title="Do application development require special skills?">
    Developers familiar with Docker and Dify development technologies can quickly develop applications for RM-01. RMinte provides complete developer documentation and SDK support.
  </Accordion>
  <Accordion title="How can application developers profit from RM-01?">
    Developers can build applications on any platform and deploy them on the RMinte platform, ultimately generating revenue by providing 'Software + Hardware' solutions to end customers.
  </Accordion>
  <Accordion title="How do enterprises evaluate RM-01's ROI?">
    Compared to traditional AI deployment, RM-01 saves 80% in initial investment, reduces operational costs by 98%, and achieves 99% total cost of ownership savings over 3 years.
  </Accordion>
  <Accordion title="Are there rental or subscription models available?">
    Yes, developers or service providers can offer device rental and DaaS (Device as a Service) subscription models to reduce enterprise initial investment barriers.
  </Accordion>
  <Accordion title="Is there a trial service available?">
    Yes, we offer a 7-day free trial service for enterprise customers, with professional technical staff providing on-site support during the evaluation period.
  </Accordion>
</Accordions>
