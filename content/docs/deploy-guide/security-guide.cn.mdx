## 应用安全分发

### 应用准备

1. **应用打包**
   ```bash
   # 将应用打包为 RM-01 格式
   rm01-dev-cli package --app-dir ./myapp --output myapp.rmap
   ```

2. **应用签名**
   ```bash
   # 使用开发者密钥签名应用
   rm01-dev-cli sign --app myapp.rmap --key developer_private.pem --output signed_myapp.rmap
   ```

3. **应用加密**
   ```bash
   # 使用渠道商公钥或设备公钥加密应用
   rm01-dev-cli encrypt --app signed_myapp.rmap --public-key distributor_public.pem --output encrypted_myapp.rmap
   ```

### 零信任应用分发

以下流程确保应用只能在授权的设备上运行：

1. **渠道商阶段**
   ```bash
   # 渠道商获取设备公钥并添加到分发系统
   rm01-distributor-cli add-device --public-key device_public.pem --device-id "RM01-12345"
   
   # 将应用重新加密为特定设备专用
   rm01-distributor-cli reencrypt --app encrypted_myapp.rmap --device-id "RM01-12345" --output device_specific_app.rmap
   ```

2. **存储卡准备**
   ```bash
   # 将应用复制到 microSD 卡
   rm01-storage-cli copy --source device_specific_app.rmap --target /mnt/microsd/apps/ --verify
   ```

3. **应用部署验证**
   - 插入存储卡到目标 RM-01 设备
   - 设备会自动验证并解密应用
   - 查看部署日志确认成功
   ```bash
   rm01-cli logs --module app-deployer --last 50
   ```

### 应用更新安全

1. **增量更新包**
   ```bash
   # 创建增量更新包（减小分发体积）
   rm01-dev-cli create-update --old-version myapp_v1.rmap --new-version myapp_v2.rmap --output update_patch.rmap
   ```

2. **更新包签名和加密**
   ```bash
   # 签名
   rm01-dev-cli sign --app update_patch.rmap --key developer_private.pem --output signed_update.rmap
   
   # 加密
   rm01-dev-cli encrypt --app signed_update.rmap --public-key distributor_public.pem --output encrypted_update.rmap
   ```

3. **安全分发更新**
   - 通过存储卡分发
   - 或通过安全更新服务器（如企业内有网络连接）

## 数据安全处理

### 数据隔离

1. **应用数据隔离**
   ```yaml
   # 应用 manifest.yaml 中的数据隔离配置
   security:
     data_isolation:
       level: "strict"
       storage:
         private: "/data/private"
         shared: "/data/shared"
       network:
         isolated: true
   ```

2. **多租户隔离**
   ```yaml
   # 系统配置中的多用户隔离
   system:
     multi_tenant:
       enabled: true
       isolation:
         storage: true
         memory: true
         process: true
   ```

### 数据加密

1. **静态数据加密**
   ```python
   from rminte.security import DataEncryptor
   
   # 初始化加密器
   encryptor = DataEncryptor()
   
   # 加密数据
   encrypted_data = encryptor.encrypt(sensitive_data)
   
   # 安全存储
   with open('secure_data.enc', 'wb') as f:
       f.write(encrypted_data)
   ```

2. **传输中数据加密**
   ```python
   from rminte.security import SecureChannel
   
   # 初始化安全通道
   secure_channel = SecureChannel()
   
   # 加密通信
   encrypted_message = secure_channel.encrypt(message, recipient_public_key)
   
   # 发送加密消息
   send_data(encrypted_message)
   ```

3. **内存数据保护**
   ```python
   from rminte.security import SecureMemory
   
   # 创建安全内存区
   secure_mem = SecureMemory(size_mb=10)
   
   # 在安全内存中处理敏感数据
   with secure_mem.context() as mem:
       mem.write(sensitive_data)
       # 处理数据
       result = process_sensitive_data(mem.read())
   
   # 退出上下文后，安全内存自动清除
   ```

### 数据生命周期管理

1. **数据留存策略**
   ```yaml
   # 应用中的数据留存配置
   data_retention:
     temp_data: "session"  # 会话结束后删除
     cache: "1d"  # 保留 1 天
     logs: "30d"  # 保留 30 天
     user_data: "persistent"  # 永久保留，除非手动删除
   ```

2. **自动数据清理**
   ```python
   from rminte.security import DataCleaner
   
   # 配置数据清理器
   cleaner = DataCleaner(
       temp_dir="/app/temp",
       retention="4h",
       secure_delete=True
   )
   
   # 注册清理任务
   cleaner.register()
   
   # 手动触发清理
   cleaner.cleanup()
   ```

3. **安全删除**
   ```python
   from rminte.security import SecureDelete
   
   # 安全删除文件（多次覆写）
   SecureDelete.file("/path/to/sensitive-file.dat")
   
   # 安全清理目录
   SecureDelete.directory("/path/to/sensitive-dir")
   ```
