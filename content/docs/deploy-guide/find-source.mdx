## Quick Start

### First-time Setup

<Steps>
  <Step>
    ### Unboxing Inspection

    Confirm the package contains all required items:
    - RM-01 Main Unit
    - CFexpress Card with pre-installed models and applications
    - MicroSD Card
    - Warranty Card
    - Quick Start Guide
    - USB-C Power Adapter
    - USB-C to Ethernet Adapter
  </Step>
  <Step>
    ### Connect Power

    Connect the USB-C power adapter to the RM-01 and power outlet.

    Ensure you're using the original power adapter for optimal performance.
  </Step>
  <Step>
    ### Start the Device

    1. Press the power button
    2. Wait for device to boot (about 1-2 minutes)
    3. Verify steady white indicator light (normal operation)

    **Status Indicator Guide:**
    - White: Running normally
    - Blue: Starting up
    - Red: Error state
  </Step>
  <Step>
    ### Device Connection

    Use the USB-C to network adapter to connect to:
    - Local area network
    - Personal devices (phone, tablet, computer)
    - Enterprise network infrastructure
  </Step>
</Steps>

### Accessing the Management Interface

<Tabs items={["Web Interface","Mobile App"]}>
  <Tab value="Web Interface">
    1. Open a browser on a computer in the same local network
    2. Enter the device IP address or hostname
       - Format: `http://Assigned-IP-Address:xxxx`
       - Where `xxxx` is the last four digits of the device serial number
    3. Log in using default credentials:
       - **Username:** `admin`
       - **Password:** Scan the QR code on the Warranty Card
  </Tab>
  <Tab value="Mobile App">
    1. Download and install the "RM-01 ALTAI Companion" app
    2. Open the app and scan the QR code on the device
    3. Follow the in-app prompts to complete the connection
    4. The app will automatically detect and connect to your RM-01 device
  </Tab>
</Tabs>

<Callout title="Security First" type="warn">
  After first login, immediately change the default administrator password:

  1. Log in to the management interface
  2. Click the "Settings" icon in the top right corner  
  3. Select "Account Security"
  4. Click "Change Password"
  5. Enter and confirm the new password
</Callout>

## Frequently Asked Questions

<Accordions type="multiple">
  <Accordion title="Device won't start">
    **Possible causes and solutions:**
    - **Power issue**: Check power adapter connection, ensure using original power supply
    - **Storage card issue**: Try removing and reinserting the storage card
    - **System issue**: Hold the power button for 10 seconds to force shutdown, then restart
  </Accordion>
  <Accordion title="Application won't start">
    **Possible causes and solutions:**
    - **Insufficient resources**: Close other running applications to free up memory
    - **Corrupted application**: Reinstall the application from the App Library
    - **Model issue**: Check if the required models are correctly loaded in Model Management
  </Accordion>
  <Accordion title="Can't connect to network">
    **Possible causes and solutions:**
    - **Network adapter issue**: Check the USB-C network adapter connection
    - **Network settings error**: Verify network configuration, especially IP address settings
    - **Firewall restrictions**: Confirm the enterprise firewall is not blocking the device connection
  </Accordion>
  <Accordion title="Poor performance">
    **Possible causes and solutions:**
    - **Multiple applications running**: Reduce the number of running applications
    - **Large model load**: Switch to a smaller or more efficient model
    - **High temperature**: Ensure good device ventilation, avoid stacking or blocking vents
    - **Insufficient storage**: Clear unnecessary data to free up storage space
  </Accordion>
  <Accordion title="Data upload fails">
    **Possible causes and solutions:**
    - **File size too large**: Check if the file exceeds the application's size limit
    - **Unsupported format**: Verify the file format is supported by the application
    - **Storage full**: Clear space or use external storage options
    - **Network interruption**: Ensure stable network connection during upload
  </Accordion>
  <Accordion title="Forgot administrator password">
    **Recovery options:**
    - Use the QR code on the Warranty Card to reset the password
    - Contact your service provider for password reset assistance
    - If available, use the hardware reset button (check user manual for location)
    - As a last resort, factory reset may be required (will erase all data)
  </Accordion>
</Accordions>
