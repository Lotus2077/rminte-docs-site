## Quick Start

### First-time Setup

<Steps>
  <Step>
    ### Unboxing Inspection
    Confirm the package contains the following items:
    - RM-01 Main Unit
    - USB-C Power Adapter
    - USB-C to Network Adapter
    - CFexpress Card with pre-installed models and applications
    - Quick Start Guide
  </Step>

  <Step>
    ### Connect Power
    - Connect the USB-C power adapter to the RM-01
    - Connect the power adapter to a power outlet
  </Step>

  <Step>
    ### Start the Device
    - Press the power button
    - Wait for the device to boot (about 1-2 minutes)
    - A steady white indicator light means the device is running normally
  </Step>

  <Step>
    ### Device Connection
    - Use the USB-C to network adapter to connect to a local network or personal device (such as a phone, tablet, etc.)
  </Step>
</Steps>

### Accessing the Management Interface

<Tabs items={['Web Interface', 'Mobile App']}>
  <Tab>
    1. Open a browser on a computer in the same local network
    2. Enter the device IP address or hostname (format: `http://rm01-xxxxx.local`, where xxxxx is the device serial number)
    3. Log in using the default credentials:
       - Username: `admin`
       - Password: see label on device bottom
  </Tab>
  <Tab>
    1. Download and install the "RM-01 Companion" app
    2. Open the app and scan the QR code on the device
    3. Follow the in-app prompts to complete the connection
  </Tab>
</Tabs>

<Callout type="warning" className="border-l-4 border-amber-500 dark:border-amber-400 bg-amber-50/80 dark:bg-amber-900/20 backdrop-blur-sm">
  After first login, immediately change the default administrator password:
  1. Log in to the management interface
  2. Click the "Settings" icon in the top right corner
  3. Select "Account Security"
  4. Click "Change Password"
  5. Enter and confirm the new password
</Callout>

## Frequently Asked Questions

<Tabs items={['Device Issues', 'Application Issues', 'Network Issues', 'Performance Issues']}>
  <Tab>
    <Callout type="error" emoji="‚ùì" className="border-l-4 border-red-500 dark:border-red-400 bg-red-50/80 dark:bg-red-900/20 backdrop-blur-sm">
      **Device won't start**
    </Callout>
    
    **Possible causes and solutions**:
    - **Power issue**: Check power adapter connection, ensure using original power supply
    - **Storage card issue**: Try removing and reinserting the storage card
    - **System issue**: Hold the power button for 10 seconds to force shutdown, then restart
  </Tab>
  <Tab>
    <Callout type="error" emoji="‚ùì" className="border-l-4 border-red-500 dark:border-red-400 bg-red-50/80 dark:bg-red-900/20 backdrop-blur-sm">
      **Application won't start**
    </Callout>
    
    **Possible causes and solutions**:
    - **Insufficient resources**: Close other running applications
    - **Corrupted application**: Reinstall the application
    - **Model issue**: Check if the required models are correctly loaded
  </Tab>
  <Tab>
    <Callout type="error" emoji="‚ùì" className="border-l-4 border-red-500 dark:border-red-400 bg-red-50/80 dark:bg-red-900/20 backdrop-blur-sm">
      **Can't connect to network**
    </Callout>
    
    **Possible causes and solutions**:
    - **Network adapter issue**: Check the USB-C network adapter connection
    - **Network settings error**: Check network configuration, especially IP address settings
    - **Firewall restrictions**: Confirm the enterprise firewall is not blocking the device connection
  </Tab>
  <Tab>
    <Callout type="error" emoji="‚ùì" className="border-l-4 border-red-500 dark:border-red-400 bg-red-50/80 dark:bg-red-900/20 backdrop-blur-sm">
      **Poor performance**
    </Callout>
    
    **Possible causes and solutions**:
    - **Multiple applications running simultaneously**: Reduce the number of running applications
    - **Large model load**: Switch to a smaller model
    - **High temperature**: Ensure good device ventilation, avoid stacking
    - **Insufficient storage space**: Clear unnecessary data
  </Tab>
</Tabs>
